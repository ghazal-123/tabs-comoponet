# interactive tabs-comoponet (without Frameworks)

A dynamic, interactive tab management system built with Vanilla JavaScript, CSS3 Flexbox, and HTML5. Features real-time tab creation with unique random color generation.  
the active tab has a different background/border.

# ✨ Key Features

- **Smooth Navigation:** Instant switching between tabs using event listeners.

- **Dynamic Creation:** Add unlimited tabs without reloading the page.

- **Randomized Themes:** Each new tab is assigned a unique background color generated via HSL logic in JavaScript.

- **Full-Height Layout:** Utilizes CSS Flexbox to ensure the content area fills the entire viewport (100vh).

- **CSS Variables:** Uses --tab-color variables for clean and maintainable styling.

# 🚀 Technologies Used

**HTML5**: For the structural layout of the calculator.  
**CSS3**: For styling, Flexbox/Grid layouts.
**JavaScript**: For the computational logic and DOM manipulation.

# 🛠️ How to Use:

1.Clone the repository: git clone **https://github.com/ghazal-123/tabs-comoponet.git**

2.Open the project: Navigate to the project folder and open index.html in your favorite web browser.
